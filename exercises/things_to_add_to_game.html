Platformer Requirements

You do not need to complete all of these. But you must add requirements to your game one at a time.AI agents do best with one clear task at a time.Small changes are easier to test and fix.Later, you may use several agents at once. If each agent changes 10 things, it becomes too hard to track what happened. When you add a requirement to git, include the requirement number in the commit message.

🧍 Character & Physics

  • 2
