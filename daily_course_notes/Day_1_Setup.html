Before You Start

The order matters.

Important: If you open Claude inside a folder, that is where Claude will work, read files, edit files, and add new files.
Do not trust the class computer for storage. Data on the computer may be deleted. At the end of class, copy your project folder into a dated folder on the course Google Drive website.

Setup Steps

Do these in order from top to bottom.

1

Open Terminal

Use Spotlight search: press Command + Space, type Terminal, then press Enter.

⌘ + Space
2

Go to Desktop first

The Desktop is easy to find, so start there.

cd ~/Desktop
3

Create your project folder

Use mkdir to make a folder named project.

mkdir project
4

Move into the project folder

This is the folder Claude should work inside.

cd project
5

Check that you are in the right place

The output should end with Desktop/project.

pwd
6

Install Claude Code

Copy and paste the official macOS install command, then press Enter.

curl -fsSL https://claude.ai/install.sh | bash

Install command checked against the official Claude Code docs.

7

Start Claude and log in

Run Claude from inside the project folder. Complete the login steps in the browser or Terminal.

claude
8

Close Claude after login

After login works, close Claude so everyone starts from the same clean point.

/exit

If that does not close it, press Control + C.

9

Start Claude again inside project

Make sure the Terminal path still shows project, then start Claude.

claude

If claude Does Not Work

This usually means Terminal cannot find Claude yet.

Add Claude to Terminal's PATH. If typing claude does not work, run these two commands. They tell Terminal where to find Claude.
A

Add the Claude folder to PATH

Copy and paste this full command into Terminal, then press Enter.

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
B

Reload Terminal settings

Run this command so Terminal reads the new PATH setting.

source ~/.zshrc
C

Return to project and try again

Go back to the project folder, then start Claude.

cd ~/Desktop/project claude

When Claude Asks for Permission

Class setup is smoother when everyone chooses the same option.

Choose Yes or Allow. When Claude asks for permission during this class setup, choose Yes or Allow each time. If there is a "Yes, and don't ask again" option, use it for the project folder.
