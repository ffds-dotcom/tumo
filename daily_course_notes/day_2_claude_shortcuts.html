Keyboard Shortcuts

These work inside Claude Code at any time — no typing required.

Shortcut What it does When to use it
Shift + Enter Add a new line without sending Writing a long prompt — press this to go to the next line instead of sending early.
↑ Recall your last message You sent a prompt and want to edit it and try again.
Ctrl + C Stop Claude immediately Claude is writing code you didn't ask for — stop it before it goes further.
Esc Clear what you are typing You started typing something and want to start over without sending.

Slash Commands

Type these at the prompt and press Enter. They always start with /.

Command What it does When to use it
/clear Wipe conversation history and start fresh Claude is confused, repeating itself, or giving you code from the wrong file.
/compact Compress history to free up context space You have been building for a while and Claude starts forgetting earlier decisions.
/help Show all available commands You forget a command name.
/model Switch between AI models Use opus for hard logic problems. Use haiku for quick simple questions.
/review Run a code review on your recent changes Before showing your work to the teacher or making your next commit.

Prompt Tricks

Add these inside any message to give Claude more information.

Trick What it does Example
@filename Attach a specific file so Claude can read it @index.html Fix the bug on line 12
! command Run a terminal command without leaving Claude ! git status
Paste error Copy the red error text from your browser console and paste it directly into your message Paste error → type "fix this" → send
