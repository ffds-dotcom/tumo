Basic Mac Shortcuts

Use these everywhere: browser, editor, notes, and Finder.

Shortcut Action When to use it
⌘ + C Copy selected text or file Save something without removing it.
⌘ + V Paste copied text or file Put copied content somewhere new.
⌘ + Z Undo the last action Fix a mistake quickly.
⌘ + Space Open Spotlight search Find Terminal, apps, or files fast.
⌘ + Tab Switch between open apps Move between Terminal, browser, and editor.
⌘ + S Save current file Save before testing code.

Terminal Navigation

These commands help you know where you are and move safely.

Command Meaning Example
pwd Print the folder you are currently inside. pwd
ls List files and folders. ls
cd Change folder. cd Desktop
cd .. Move one folder up. cd ..
clear Clean the Terminal screen. clear
history Show commands you typed before. history

Unix File Commands

Practice in a test folder before using real class files.

Command Meaning Example
mkdir Make a new folder. mkdir day1
touch Create an empty file. touch notes.txt
cat Show the contents of a text file. cat notes.txt
cp Copy a file. cp notes.txt backup.txt
mv Move or rename a file. mv notes.txt day1.txt
rm Remove a file. Be careful: it can delete permanently. rm old-file.txt
