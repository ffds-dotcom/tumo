Git Basics

Git helps you save your work while you code. Today you will use Git only on this computer.

student@tumo project % git init
Git is ready in this folder
student@tumo project % git add index.html
student@tumo project % git commit -m "Start project"
Goal

Use Git to save your work while you build a project.

Practice

Hover over each command, read what happens, then try it in a test project.

Class Rule

Only commit work you can explain. Use a short, clear message.

4 Git Commands

Use only these commands today.

Command What it does Try this
git init Turn Git on in this folder. git init
git add Tell Git which file to save next. git add index.html
git commit Save your work with a short note. git commit -m "Start project"
git rm Delete a file that Git knows about. git rm old.html

Save Your Work

Do these steps in order.

1

Turn Git on

Run this inside your project folder. Do this one time.

git init
2

Pick a file

Use git add before you save.

git add index.html
3

Save with a message

Use simple words to say what changed.

git commit -m "Start project"
On this computer: Today Git saves your work here. You do not need an online account.

Delete a File

Use this only for a file Git already knows.

Think before you delete. The file leaves the folder. Then use git commit to save that change.
1

Delete the file

Use the real file name.

git rm old.html
2

Save the delete

Write a short message that says what you deleted.

git commit -m "Remove old page"
